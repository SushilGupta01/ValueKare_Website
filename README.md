
  # Website Redesign with New Tech

A modern, high-performance website for ValueKare Group, built to showcase healthcare outsourcing, hospital operations consulting, and asset management services with enterprise-level design and smooth animations.

Premium enterprise UI/UX (healthcare-grade)

Smooth motion & micro-interactions

Fast loading with Vite

Scalable component architecture

React 18 – Component-based UI

TypeScript – Type-safe development

Vite – Ultra-fast bundler

Tailwind CSS – Utility-first styling

shadcn/ui – Accessible UI components

motion/react – Fluid animations

Lucide Icons – Minimal iconography

EmailJS – Contact form integration
